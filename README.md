## Setup

### Backend

- Copy `.env.example` to `.env` and fill in the required values.
- Instal docker if not installed on system
- Run `docker compose up -d` to start the application.

### Mobile

- For the mobile run `yarn` or `npm install`
- Copy `.env.example` to `.env` and fill in the required values.
- run `yarn start`
